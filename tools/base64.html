Base64

Base64 is an encoding scheme used to represet binary data usually in an ASCII (American Standard Code for Information Interchange) string format. Base64 proves itself to be very useful in transporting data over media that are incompatible with binary data, such as email, web pages, etc. It is also used to store data in database, however base64 encoding does not provide any security, it's just convenient for storing and transmitting data.
